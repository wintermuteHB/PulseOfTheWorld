# Contributing

Contributions are welcome! Please open an issue first to discuss what you'd like to change.

## Getting Started

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-idea`)
3. Commit your changes
4. Push to the branch
5. Open a Pull Request

## Code Style

- Vanilla JS — no frameworks unless absolutely necessary
- CSS custom properties for theming
- Semantic HTML
- Mobile-first responsive design

## Adding Metrics

To add a new pulsing metric:

1. Find a reliable data source (UN, WHO, World Bank preferred)
2. Calculate the per-second rate
3. Add an entry to the metrics configuration
4. Include source attribution
